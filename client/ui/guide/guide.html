User Guide

This is an example user guide window implemented as a UI component. Press I to dismiss/reopen this window.

Commands

  • Type /nick name to set a nickname
  • Type /youtube video-id to open a youtube video
  • Type /go url to open web page
  • Type /logout to simulate logout
  • Press I toggle this view
  • Press ENTER to start chat
  • Press / to start a slash command

    Goals

    The Goal of this UI framework is to make it easy to prototype / develop self contained UI components. In its current form, it is not a framework for use within a game client, though it may evolve in that direction over time.
